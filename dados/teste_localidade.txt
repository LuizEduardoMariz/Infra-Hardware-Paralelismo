INTERPRETAÇÃO:
  • A matriz é armazenada em memória 'row-major' (linhas contíguas).
  • Loop A acessa endereços vizinhos → o cache traz os próximos
    elementos de graça (cache hit).
  • Loop B salta de linha em linha → o cache é invalidado a cada
    acesso (cache miss).
  • Mesma quantidade de cálculo, MUITA diferença de tempo.
  • Conclusão: o gargalo não é a CPU — é trazer dados até ela.
